"¿Cómo veo y/o imprimo documentos PostScript?," preguntas? ¡Fácil! Simplemente descarga algún software gratuito en:www.cs.wisc.edu/~ghost.

Allí encontrarásGSview and Guión fantasma, dos programas necesarios para mostrar e imprimir archivos Postscript (incluso mostrarán e imprimiráncomprimidoarchivos PostScript!). Estos programas también muestran y dan formato a archivos Adobe PDF como beneficio adicional. Versiones disponibles para Windows, OS/2 y Linux.

Descargar archivos fuente para este volumen

Aproximadamente 24 megas




Aproximadamente 1 megabyte

Para "compilar" estos archivos fuente en un formato visible, necesitará los siguientes programas (todos disponibles gratuitamente en Internet):

  • Hacer, una utilidad de gestión de proyectos originalmente pensada como herramienta de programación, pero útil para gestionar casi cualquier tipo de proyecto informático compuesto por muchos archivos.Si no puede obtener una copia de Make para su sistema informático, puede arreglárselas con un poco de habilidad y algunos archivos por lotes (también conocidos como scripts de shell). El "Makefile" maestro en este directorio se puede leer con un editor de texto o procesador de textos y contiene todas las instrucciones realizadas por las otras utilidades.
  • Sed(Significa Stream EDitor), una utilidad común de UNIX para realizar comandos de búsqueda y reemplazo en archivos de texto. Requerido para convertir el código fuente SubML a HTML, TeX, LaTeX y otros formatos.¡Esto es todo lo que necesitas para generar resultados HTML!
  • LaTeX2e, un sistema de formato de documentos diseñado como una extensión de TeX, el excelente sistema de procesamiento de textos de Donald Knuth. También puede arreglárselas simplemente con TeX, pero su salida impresa no se verá tan bien y carecerá de tabla de contenido y entradas de índice.

Si opta por el más pequeño de los dos archivos (EXPtiny.tar.gz), también necesitará un conjunto de utilidades de manipulación gráfica lanzadas como un paquete llamadoImagenMagia. Específicamente, la utilidad que necesitará se llamamogrificar. El mayor de los dos archivos fuente contiene todas las imágenes gráficas en dos formatos: PostScript encapsulado (*.eps) y JPEG (*.jpg). Esto lo convierte en un archivo grande. El archivo fuente más pequeño solo contiene PostScript encapsulado para diagramas esquemáticos e imágenes JPEG para fotografías. Esto lo convierte en un archivo mucho más pequeño, pero requiere que usted realice alguna conversión de imagen por su parte. Si tiene acceso a otro software de manipulación de imágenes capaz de convertir cientos de archivos con un comando por lotes, no tendrá que usar ImageMagick.
